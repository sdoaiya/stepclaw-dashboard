## 原版核心功能清单 (必须恢复)

### 1. 视觉设计
- [ ] 玻璃态 (Glassmorphism) 卡片效果
- [ ] 渐变背景 + 动态光效
- [ ] 毛玻璃模糊效果
- [ ] 精致的阴影和边框

### 2. 布局结构
- [ ] 三栏布局 (左: 统计/部门, 中: 员工, 右: 任务/动态)
- [ ] 响应式适配 (PC端三栏, 移动端单栏)
- [ ] 固定头部导航

### 3. 功能模块
- [ ] **部门筛选**: 点击部门高亮对应员工
- [ ] **实时统计**: 在线人数、Token消耗、任务数动态更新
- [ ] **员工卡片**: 头像、状态、当前任务、部门标识
- [ ] **任务列表**: 优先级标识、负责人、时间
- [ ] **动态Feed**: 实时活动流
- [ ] **主题切换**: 明暗模式

### 4. 动效
- [ ] 卡片悬停效果
- [ ] 状态切换动画
- [ ] 数据更新过渡
- [ ] 页面加载动画
