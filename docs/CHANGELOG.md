# 更新日志

## [3.0.1] - 2024-03-28

### 修复
- 🐛 **MSS内存泄漏修复**: 修复了监控面板自动刷新导致的内存泄漏问题
  - 使用 `useRef` 存储interval引用，确保正确清理
  - 修复了useEffect依赖数组导致的无限循环问题
  - 添加ECharts实例清理，防止图表内存泄漏
- 🐛 **性能优化**: MetricCard组件添加React.memo减少不必要的重渲染

### 改进
- ⚡ **构建优化**: Vite配置优化
  - 添加代码分割策略，按vendor/ui/charts/utils分包
  - 启用Terser压缩，移除console和debugger
  - 设置GitHub Pages部署基础路径
- 🚀 **CI/CD**: 添加GitHub Actions自动部署工作流
